HL7 Version 2 Table Namespaces

[%wgt fhir%] Work GroupMaturity Level: N/AStandards Status:External

These system namespaces are defined for use in FHIR when representing codes that were defined as part of the HL7 v2 tables in FHIR instances. So if you are representing the code "M" from table 0001 in HL7 v2.3.1 in a FHIR instance, then you would use the system "http://terminology.hl7.org/CodeSystem/v2-0001".

There are a number of subtle but serious issues with using tables defined for HL7 v2 as code systems, so some of these tables are mapped to a different namespace for each version. All of these namespaces are subject to further change as additional review of the tables proceeds.
