D.23.5.0 U.S. Data Access Framework (DAF) DAF Specimen Profile

D.23.5.0.1 Scope and Usage

This profile sets expectations for use of the Specimen resource to record a specimen associated with a patient within the DAF FHIR IG. This profile identifies which core elements, extensions, vocabularies and value sets must be Supported by DAF actors. For the definition of Supported please refer to DAF FHIR IG.

D.23.5.0.2 Meaningful Use Data Element Mapping

The DAF Specimen profile mapping does not have a direct mapping to Meaningful Use elements. The DAF team added this profile in response to DSTU 2 ballot comments. For an overview of how the DAF profiles and data elements were derived from ONC 2014 Edition S&CC please refer to DAF FHIR IG data mapping section.

D.23.5.0.3 Boundaries and Relationships

This profile relies on the following other profiles:

  • DAF-Patient which defines DAF Patient profile is used by DAF-Specimen profile to refer to patients.
  • DAF-ResultObs which defines DAF Results and references this DAF-specimen profile. Specimen profiles will be accessed as part of the DAF-ResultObs profiles.
