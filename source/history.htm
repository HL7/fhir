Version History

Version Major Changes
0.07
(After 2nd Draft For Comment)
  • Change to resource references: id -> url, true relative references, remove version link
  • Introduce contained resources
  • Move primitive values into attributes instead of text. Remove extension.ref, and put extensions in the place they apply
  • Remove special case for data type properties: now they have @id, and the json representation is the same
  • Changes to search: rename special parameters, introduce paging, clarify date searches
  • Change to updates: /[resource]/ is now search, rename to history operation, introduce global updates operation
  • Clarify server obligations on batch & add search specifier
  • Move Person into Patient/Practitioner (which used to be "agent")
  • Rename LabReport to DiagnosticReport & use Observation for atomic data
  • Many new resources, including Order/OrderResponse, Picture, CarePlan
