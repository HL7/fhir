* Dosage.simple has been removed (FHIR-54651, https://jira.hl7.org/browse/FHIR-54651)
  Dosage.simple was redundant since a simple dosage can be expressed as a single step.
  Removed dosdet-1 invariant; added invariant that step.safety shall not be populated when only one step is present. Updated documentation and examples.
  Files: source/datatypes/dosagedetails.xml, source/dosage.html, source/dosage-examples.html

* Fix entered-in-error definitions for MedicationStatement and MedicationRequest (FHIR-54903, https://jira.hl7.org/browse/FHIR-54903)
  Corrected the misleading entered-in-error status definitions to simply state the resource was entered in error and is not valid.
  Files: source/medicationstatement/codesystem-medication-statement-status.xml, source/medicationrequest/codesystem-medicationrequest-status.xml

* Clarify MedicationStatement.reason vs .relatedClinicalInformation (FHIR-54451, https://jira.hl7.org/browse/FHIR-54451)
  Updated .reason definition to clarify it covers what leads to taking (or not taking) a medication. Added clarification to .relatedClinicalInformation that it is not the actual indication - that belongs in .reason.
  Files: source/medicationstatement/structuredefinition-MedicationStatement.xml

* Change Dosage.site from CodeableConcept to CodeableReference(BodyStructure) (FHIR-54848, https://jira.hl7.org/browse/FHIR-54848)
  Allows referencing BodyStructure for richer site descriptions (laterality, location qualifiers) while keeping the same coded binding.
  Files: source/datatypes/dosage.xml

* Make DosageDetails.renderedInstruction required (1..1) (FHIR-54014, https://jira.hl7.org/browse/FHIR-54014)
  Ensures dosage intent is always communicated. Updated definition and comment.
  Files: source/datatypes/dosagedetails.xml, source/medicationdispense/medicationdispense0302.xml, source/medicationdispense/medicationdispense0305.xml, source/medicationdispense/medicationdispense0306.xml, source/medicationdispense/medicationdispense0319.xml, source/medicationrequest/medicationrequest0302.xml, source/medicationrequest/medicationrequest0303.xml, source/medicationrequest/medicationrequest0317.xml, source/medicationrequest/medicationrequest0321.xml, source/medicationrequest/medicationrequest0339.xml, source/medicationstatement/medicationstatementexample4.xml
