FHIR Management GroupMaturity Level: N/AStandards Status: Normative

Formal Name:HL7 FHIR® R4 Infrastructure, Release 1
ANSI Designation:ANSI/HL7 FHIR® R4 INFRASTRUCTURE R1-2019
Date of Approval:2019-03-22

This ANSI Normative Standard contains the Infrastructure parts of the FHIR specification - the core framework on which FHIR solutions depend (Abstract base types, data types, formats, the RESTful API, and content and typing rules).

This standard uses the ANSI FHIR Conformance standard (for CapabilityStatement).
