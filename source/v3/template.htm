HL7 Version 3 Code System Namespaces

These system namespaces are defined for use in FHIR when representing codes that were defined as part of the v3 code systems. So if you are representing the code "M" from the code system AdministrativeGender in a FHIR instance, then you would use the system "http://hl7.org/fhir/v3/AdministrativeGender" (note that the namespace is case sensitive).

All of these namespaces are subject to further change.
