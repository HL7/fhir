FHIR Conformance Package

FHIR Management GroupMaturity Level: N/AStandards Status: Normative

Formal Name:HL7 FHIR R4 Terminology & Conformance, Release 1
ANSI Designation:ANSI/HL7 FHIR R4 TERMINOLOGY R1-2019
Date of Approval:2019-08-09

This ANSI Normative Standard contains the Conformance parts of the FHIR specification - the terminology infrastructure and the base resources that define FHIR implementations and that represent how the specification, and conformance to it, is defined.

This standard uses the ANSI FHIR Infrastructure standard.
